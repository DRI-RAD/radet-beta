# RADET Examples

This folder contains example notebooks and scripts demonstrating RADET model usage and related analyses.

## Notebooks

- **radet_single_image.ipynb** — Compute RADET ET for a single Landsat image
- **radet_collection_interpolate.ipynb** — Build an ET image collection and perform temporal interpolation

## Scripts

- **eecu_analysis.py** — Analyze Earth Engine Compute Unit (EECU) usage across OpenET models

## Data

- **eecu_data/** — Raw EECU input data
- **eecu_output/** — Generated EECU analysis results (CSVs, plots)
